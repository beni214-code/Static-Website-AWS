# Static-Website-AWS
This project is a beginner-friendly deployment of a static website using Amazon S3 and CloudFront. I wanted hands-on experience in hosting and content delivery, so this project helped me put that knowledge into practice.

#  Architecture

Amazon S3: Stores the website’s static HTML files.

Amazon CloudFront: Distributes the content globally.

# Steps I Took

1. Created basic HTML files (index and error pages).

2. Set up an S3 bucket with static website hosting enabled.

3. Configured bucket policy for public read access to website files.

4. Created a CloudFront distribution pointing to the S3 website endpoint.

5. Tested the CloudFront domain to ensure the site is available globally.

# Cost Estimate
With minimal traffic and storage, this should stay under a few dollars a month. I set a budget to monitor costs.
